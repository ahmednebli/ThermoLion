## [0.1.0] - 2025-12-02

### Added

- Initial public release of the `thermolion` package.
- `ThermoLion` optimizer implemented for PyTorch.
- Basic tests to verify the optimizer runs and decreases a simple loss.
- Packaging configuration (`pyproject.toml`), README, and MIT license.
